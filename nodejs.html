1-Intro

  1. is an open source server environment
  2. is free
  3. runs on various platforms (Windows, Linux, Unix, Mac OS X, etc.)
  4. uses JavaScript on the server
  5. uses asynchronous programming!
  6. A common task for a web server can be to open a file on the server
  7. and return the content to the client.

how PHP or ASP handles a file request:

  1. Sends the task to the computer's file system.
  2. Waits while the file system opens and reads the file
  3. Returns the content to the client.
  4. Ready to handle the next request.

how Node.js handles a file request

  1. Sends the task to the computer's file system.
  2. Ready to handle the next request.
  3. When the file system has opened and read the file, the server returns the content to the client.
  4. Node.js eliminates the waiting,and simply continues with the next request.
  5. Node.js runs single-threaded
  6. non-blocking
  7. asynchronous programming
  8. Node.js can generate dynamic page content
  9. can create, open, read, write, delete, and close files on the server
  10. Node.js can collect form data
  11. Node.js can add, delete, modify data in your database
